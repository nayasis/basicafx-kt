# basicafx

basic JavaFx library bases on Kotlin.

[![Maven Central](https://img.shields.io/maven-central/v/io.github.nayasis/basicafx-kt.svg)](https://central.sonatype.com/artifact/io.github.nayasis/basicafx-kt)

## Dependency

### gradle (build.gradle.kts)

```kotlin
dependencies {
    implementation("io.github.nayasis:basicafx-kt:0.2.3")
}
```

### maven (pom.xml)

```xml
<dependency>
    <groupId>io.github.nayasis</groupId>
    <artifactId>basicafx-kt</artifactId>
    <version>0.2.3</version>
</dependency>
```
